# 万能导入 V2 — 大模型调用说明

## 使用模型

| 配置项 | 说明 |
|--------|------|
| 模型 | DeepSeek Chat（`deepseek-chat`），兼容 OpenAI API |
| API 地址 | `https://api.deepseek.com/v1` |
| SDK | `@ai-sdk/openai` + Vercel AI SDK (`generateObject`) |

## AI 介入环节

### 文件智能分析 → 自动生成解析规则

**触发时机**：用户上传出库单文件后点击"AI 智能分析"

**流程**：
1. 读取文件前 50 行内容，生成文本预览
2. 调用 AI，使用 `generateObject` + zod schema 生成结构化解析规则
3. AI 返回：规则名称、文件类型、表头行号、数据起始行、字段映射、是否需要复合拆分/矩阵转置/尾部提取
4. 前端展示 AI 结果，用户确认后保存规则并自动执行解析

**Prompt 设计**：
```
你是一位物流出库单解析专家。请分析用户上传的文件样本，生成一个通用的解析规则配置。

目标字段：external_code(外部编码), store_name(门店), receiver_name(收件人),
receiver_phone(电话), receiver_address(地址), sku_code(SKU编码),
sku_name(SKU名称), quantity(数量), spec(规格), remark(备注)

规则能力：
- 识别干扰头部行、表头行、数据起始行
- 识别合并单元格、矩阵转置格式
- 识别复合单元格（物品名x数量）
- 识别尾部收货人信息
- 识别卡片式记录
```

**结构化输出**：通过 zod schema 确保 AI 返回格式正确，`guessed` 字段列出推测项供用户确认。

## 安全原则

- AI 分析结果仅供用户确认，不会自动执行
- AI 服务不可用时，用户可手动创建规则
- temperature=0.2 保证输出稳定性
